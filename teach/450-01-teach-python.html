01 Teach : Python Practice

Objective

Practice elements of Python that will be useful in your programming this semester.

Instructions

In groups of 2-3, work together to complete the following sections of the CodeAcademy Python Tutorial:

  • Unit 7: Lists and Functions
  • Unit 10: Advanced Topics in Python
  • (Extra Credit) Unit 11: Introduction to Classes

For each section, complete the first lesson in that unit (you do not need to do the second, practice lesson, nor the quizzes/projects). After class, you are welcome to complete any work that you did not finish during class.

When complete, report on the portions of each section you completed with the accompanying I-Learn quiz.
